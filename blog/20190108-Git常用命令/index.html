在线仓库

在线的 Git 仓库，国外比较出名的有github、gitlab等，国内的有码云gitee、coding等。

  • github：不多说了，09 年 1 月 7 号新增了免费仓库（仓库协作者 3 个以内无限免费），微软太强大了，做得好！
  • gitee(码云)： 国内的代码管理平台，广告有点多，但国内使用速度快，如果嫌github速度慢的话，可以使用它。

IDE（编辑器）

现在很多的IDE或者编辑器都自带了 git 的可视化操作，以后可以不用死记 git 的命令了，直接使用编辑器的可视化操作。下面简单介绍一下。

  • WebStorm

    WebStorm 的 git 操作非常强大，用了之后都不想使用 git 命令了。

右键目录栏里面的文件/文件夹，选中 git ，然后就是各种 git 的操作了（add, commit, revert(回退)等等)，非常方便。

甚至可以不用先去创建仓库，直接就分享到 github 中，点击VCS - Import into Version Control - Show Project on Github即可。

  • VSCode

VSCode 是 MicroSoft 开源的编辑器，是现在最流行最好用的免费的编辑器之一，插件系统非常强大，装好插件之后写代码就是一个“爽”字。里面有多款 git 插件，非常强大。当然它本身也集成了 git 功能。

编辑器左侧有一个Source Control源代码管理，也可以进行可视化操作，再配合一些 git 的插件，非常好用，而且支持中文，这里就不过多介绍了，谁用谁知道。

总结

上面的命令可以满足大部分的日常开发需求了。如果还有其他需求，可以去官方网站查看相关文档。
