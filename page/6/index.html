声明提升

JavaScript 使用变量的时候必须要声明，一般在使用前声明，也可以在使用后声明。

JavaScript 程序运行的时候，会自动把变量的声明提升到最前面去，所有变量的声明可以在使用后声明；但是不会把赋值提升上去，比如使用后声明的a=1，在使用时不会获取 a 的值是 1，而是undefined，这样没什么意义。建议使用变量还是提前声明比较好。
